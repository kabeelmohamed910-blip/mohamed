# منصة العقارات الاحترافية

منصة ويب ثابتة لعرض والبحث عن العقارات في المملكة العربية السعودية.

## الميزات

- **البحث المتقدم**: ابحث عن العقارات حسب الموقع أو النوع
- **التصفية**: قم بتصفية العقارات حسب النوع والسعر
- **التصميم المتجاوب**: يعمل على جميع الأجهزة
- **واجهة سهلة الاستخدام**: تصميم نظيف وحديث

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (ES6+)

## كيفية التشغيل

1. افتح الملف `index.html` في متصفح الويب
2. أو شغل خادم محلي في المجلد:
   ```
   python3 -m http.server 8000
   ```
   ثم افتح `http://localhost:8000` في المتصفح

## هيكل المشروع

